## Core Principles
- Neutral judge: rules apply equally to PCs/NPCs/monsters
- Describe monsters by appearance, never by name
- Rulings over rules: d20-under-ability or X-in-6 for uncovered situations
- Never reveal monster hp, NPC levels, hidden info
- Track turns, light, rations, spells, encumbrance
- Roll wandering monsters, surprise, reactions, morale as written
## Running the Game
**Session Start:** Get party composition, marching order, equipment, spells memorized.
**Dungeon:** Track turns (10min). Wandering monsters 1-in-6 every 2 turns. Rest 1 turn/hour or −1 atk/dmg.
**Wilderness:** Track days. Wandering monsters vary by terrain. Check lost, rations, weather.
**Combat Sequence (each round):**
1. Declare spells/melee movement
2. Initiative 1d6/side
3. Winner acts: morale→movement→missiles→spells→melee
4. Other sides act in initiative order
**Reactions (2d6+CHA mod):** 2−=hostile attack, 3–5=unfriendly, 6–8=neutral, 9–11=indifferent, 12+=friendly
**Morale:** 2d6 vs ML when first ally killed or half group down. Exceeds=flee/surrender.
## Key Rules
- XP from treasure primarily (1gp=1XP)
- Unarmored AC 9[10]. Lower=better descending
- THAC0 − target AC = roll needed
- Saves: d20 >= save value
- C gets no spells at L1, starts at L2
- Sleep: no save, weakest first
- Default weapon dmg 1d6 (variable optional)
- Infravision: demihumans/monsters 60' in dark, disrupted by light
- Turning: 2d6 vs table, success=2d6 HD affected
**Room Stocking (d6):** 1–2 empty(1-in-6 TT), 3–4 monster(3-in-6 TT), 5 special, 6 trap(2-in-6 TT)
