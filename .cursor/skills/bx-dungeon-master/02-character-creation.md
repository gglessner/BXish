# Character Creation
## Ability Modifiers
**STR:** 3=−3/1-in-6, 4–5=−2/1-in-6, 6–8=−1/1-in-6, 9–12=0/2-in-6, 13–15=+1/3-in-6, 16–17=+2/4-in-6, 18=+3/5-in-6 (melee atk+dmg / open doors)
**INT:** 3=broken/illiterate, 4–5=native/illiterate, 6–8=native/basic, 9–12=native/literate, 13–15=+1lang, 16–17=+2lang, 18=+3lang
**WIS:** 3=−3, 4–5=−2, 6–8=−1, 9–12=0, 13–15=+1, 16–17=+2, 18=+3 (magic saves, not breath)
**DEX:** 3=−3AC/−3missile/−2init, 4–5=−2/−2/−1, 6–8=−1/−1/−1, 9–12=0, 13–15=+1/+1/+1, 16–17=+2/+2/+1, 18=+3/+3/+2
**CON:** 3=−3, 4–5=−2, 6–8=−1, 9–12=0, 13–15=+1, 16–17=+2, 18=+3 (hp per HD, min 1hp/HD)
**CHA:** 3=−2react/1max/4loy, 4–5=−1/2/5, 6–8=−1/3/6, 9–12=0/4/7, 13–15=+1/5/8, 16–17=+1/6/9, 18=+2/7/10
## Prime Requisite XP Mod
3–5=−20%, 6–8=−10%, 9–12=none, 13–15=+5%, 16–18=+10%
## Alignment
L=Law, N=Neutrality, C=Chaos. Alignment language: secret signs, unwritten, lost if changed.
## Languages
Common + alignment + INT bonus. Bonus list: Bugbear, Doppelgänger, Dragon, Dwarvish, Elvish, Gargoyle, Gnoll, Gnomish, Goblin, Halfling, Harpy, Hobgoblin, Kobold, Lizard Man, Medusa, Minotaur, Ogre, Orcish, Pixie, Human Dialect
## Creation Steps
1. Roll 3d6 in order: STR, INT, WIS, DEX, CON, CHA
2. Choose class (note min ability reqs)
3. Optionally lower STR/INT/WIS (not below 9) to raise prime req (2:1)
4. Note modifiers from tables
5. THAC0 19[0] at L1 for all
6. Note saves & class abilities; referee assigns starting spells
7. Roll HP: class HD + CON mod (min 1hp)
8. Choose alignment: L/N/C
9. Languages: Common + alignment + INT bonus
10. Buy equipment: 3d6×10 gp
11. AC = armor + DEX mod (unarmored 9[10])
12. Level 1, 0 XP
13. Name character
