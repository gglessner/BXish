# Core Rules
## Dice
d4/d6/d8/d10/d12/d20=single die. 3d6=roll 3d6 sum. 3d6+2=sum+2. d100(d%)=2d10 tens+ones, 00=100. "Natural"=before mods.
## Resolution
X-in-6: 1d6 ≤ chance = success. Saves: d20 ≥ value. Ability check: d20 ≤ score.
## Classes
C, F, MU, T (human); D, E, H (demihuman). 7 total.
## Scope
C spells 1–5, MU spells 1–6. 200+ monsters, 150+ magic items. Dungeon/wilderness/waterborne exploration. Strongholds & domains. 100% B/X 1981 compatible.
