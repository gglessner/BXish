# Advancement & Wealth
## XP
Sources: 1) Treasure recovered (1gp=1XP, primary), 2) Monsters defeated.
PR XP Mod: 3–5=−20%, 6–8=−10%, 9–12=none, 13–15=+5%, 16–18=+10%.
Multi-prime: E(INT+STR both 13+=+5%, INT 16+ & STR 13+=+10%), H(DEX/STR one 13+=+5%, both 13+=+10%).
## Leveling Up
1. Check class progression table
2. Update saves, THAC0, spells, abilities
3. Roll new HD (class die) + CON mod for hp
4. Levels with * = flat hp, no CON mod
Max 1 level gain/session. Excess XP capped at 1 below next level.

## Wealth
Coins: 1pp=5gp, 1gp=10sp=100cp, 1ep=½gp.
Starting: 3d6×10 gp. Inheritance: 10% tax, new L1 heir only, one per player.
Domains: 10gp tax/settler/year.
