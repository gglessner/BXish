# Hired Help
## Retainers
NPCs for adventures. Limit: CHA max retainers. Level ≤ hiring PC. Not slaves.
**Wages:** Guaranteed fee + ½ share treasure minimum. Hirer pays food/lodging/gear.
**Applicant Reaction (2d6):** 2−=ill will(−1 future), 3–5=refused, 6–8=reroll, 9–11=accepted, 12+=accepted+1 loyalty. Mods: CHA react, ±1/2 offer, −1/2 bad rep.
**XP:** Same as PCs but −50% penalty. L0 retainer must choose class.
**Loyalty:** Base=CHA loyalty score. Check 2d6 ≤ loyalty. Fail in peril=flee. Fail after adventure=won't return.
